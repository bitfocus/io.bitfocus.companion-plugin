Companion Plugin - General Configuration
Connection Settings

The Satellite API allows this plugin to connect to Companion over a direct TCP connection. This is the recommended protocol for Companion 4.3 and later.

1. Open Companion's web GUI and go to the Settings tab.
2. Scroll down to the Satellite API section and ensure it is enabled.
3. Enter the host and port below. If Companion is running on the same machine, use 127.0.0.1.

Default port is 16622 — most users should not change this.

The Satellite API also supports WebSocket connections. Use this if TCP is blocked or unavailable in your environment. Requires Companion 4.3 or later with the Satellite API enabled.

Enter the full WebSocket URL below. If Companion is running on the same machine, the default is ws://127.0.0.1:16623.

Default port is 16623 — most users should not need to change the port number.

Each instance of this plugin is identified by a unique Device ID when connecting via the Satellite API. This allows multiple Stream Deck Plugin instances to connect to the same Companion installation simultaneously. You should only need to change this if you have a conflict with another instance, or you want to restore the id from a previous installation of the Elgato software.

The legacy protocol uses a proprietary WebSocket JSON connection supported by Companion 2.4 and later.

This protocol is deprecated and will be removed in a future Companion release. Switch to Satellite TCP if you are running Companion 4.3 or later.

Follow these steps to enable the legacy connection in Companion:

1. Open Companion's web GUI and go to the Settings tab.
2. Scroll down to the Surfaces section.
3. Disable the option "Enable connected Streamdecks". When disabled, support for the Elgato software Plugin will be enabled (requires a Companion restart).

Note: disabling this option means Companion will no longer communicate with locally connected Stream Deck hardware on the same machine. Satellite instances are unaffected.

Default port is 28492 — most users should not change this.
