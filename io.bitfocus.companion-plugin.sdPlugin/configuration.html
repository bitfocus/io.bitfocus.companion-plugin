Companion Plugin - General Configuration
IP/Port Settings

The Companion Plugin communicates with Bitfocus Companion via a proprietary websockets protocol.

Follow these steps to ensure the plugin can connect to Companion:

1. Open Companion's web GUI and go to the "Settings" tab.
2. Scroll down to the "Surfaces" section.
3. Disable the option, "Enable connected Streamdecks". When disabled, support for the Elgato software Plugin will be enabled (Requires Companion restart).

Please note that by disabling this option, Companion will lose the ability to communicate with any locally connected Streamdeck devices on the same computer running Companion as it will assume this Elgato software is in control. Satellite instances will still work fine.
